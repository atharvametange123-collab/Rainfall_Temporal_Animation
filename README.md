# Rainfall_temporal_animator 
An automated CI/CD Pipeline that converts raw dimensional rainfall data into animation. 


What it does ?
This tool ingests daily gridded rainfall data (NetCDF format) and automatically renders a continuous, color-mapped spatial animation (GIF) tracking hydrologic changes over time.

You do not need to install Python to use this tool!
1. Click `Add file` > `Upload files` at the top of this repository.
2. Drag and drop your `.nc` dataset (e.g., IMD gridded data) and click Commit.
3. Click the **Actions** tab to watch the cloud server process your data.
4. When the run completes, download your animated GIF from the **Animations** section!

Techstack 
Languages: Python(Xarray, Pandas, Matplotlib)
